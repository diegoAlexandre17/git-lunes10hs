REMOTE (Repositorio Remoto)

	Es una direccion donde podemos subir o descargar modificaciones con respecto a nuestro
	repositorio. Se utiliza como punto de referencia asi como tambien para la copia del
	contenido de nuestro repositorio.

SITIOS DE REPOSITORIOS:
	https://github.com
	https://gitlab.com
	https://bitbucket.org

GIT COMANDOS:
	
	git remote = lista los remotos cargados en el repositorio
	git remote add <alias> <url> = carga un remoto
	git remote remove <alias> = remueve el remoto de la lista
	git remote rename <original> <nuevo> = cambia el nombre del remoto
